# Data Visualization: estilização de tabelas com Python

Este projeto é focado na manipulação avançada e visualização de dados utilizando a biblioteca **Pandas**. O objetivo é transformar DataFrames brutos em tabelas estéticas, profissionais e informativas, utilizando propriedades de estilo nativas e CSS customizado.

---

## Objetivos do Projeto

- **Domínio da Classe `Styler`**: Utilização da propriedade `df.style` para manipulação visual de DataFrames.
- **Built-in Styles**: Aplicação de estilos prontos do Pandas para realçar informações (máximos, mínimos e gradientes).
- **Customização com CSS**: Criação de estilos próprios utilizando sintaxe CSS para controle total da aparência das tabelas.
- **Apresentação de Dados**: Construção de tabelas adaptadas para diferentes contextos de negócios e relatórios.

## Tecnologias e Conceitos Utilizados

- **Python**: Linguagem base.
- **Pandas**: Biblioteca principal para análise e tratamento de dados.
- **CSS (Cascading Style Sheets)**: Aplicado via Python para definir cores, fontes, bordas e alinhamentos.
- **Jupyter Notebook**: Ambiente utilizado para renderização das tabelas estilizadas.

## Requisitos e Dependências

Para executar os scripts deste projeto e visualizar as tabelas estilizadas, você precisará das seguintes bibliotecas:

* **Python 3.x**
* **Pandas**: Para manipulação e criação da classe `Styler`.
* **Jinja2**: Necessário para que o Pandas consiga renderizar as estilizações em HTML/CSS.
* **Matplotlib**: Utilizado internamente pelo Pandas para alguns mapas de cores (colormaps).
